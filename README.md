
# Strava API Running Dashboard

A running dashboard that utilises Strava's API to add more customisation to Strava analytics
(Learning Project)


# Setup

1. Clone the repository
2. Create virtual environment: `python3 -m venv venv`
3. Activate: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Generate Swagger client: `./setup_swagger.sh`
6. Create `.env` file with your Strava credentials
7. Run: `python app.py`
